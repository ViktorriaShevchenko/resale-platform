# Проект "Доска объявлений"

##  Описание проекта
Backend-часть сайта для размещения объявлений с возможностью регистрации,
авторизации, создания и редактирования объявлений, добавления комментариев.

##  Участники проекта
- Шевченко Виктория (https://github.com/ViktorriaShevchenko) - разработчик

##  Используемые технологии

### **Основные фреймворки и библиотеки:**
- **Java 11** - язык программирования
- **Spring Boot 2.7.15** - основной фреймворк
- **Spring Web** - REST API
- **Spring Data JPA** - работа с базой данных
- **Spring Security** - аутентификация и авторизация
- **PostgreSQL** - основная база данных
- **H2 Database** - база данных для тестирования

### **Работа с данными:**
- **Liquibase** - миграции базы данных (версионирование схемы)
- **MapStruct 1.5.5** - маппинг между Entity и DTO
- **Lombok** - сокращение шаблонного кода

### **Документация:**
- **OpenAPI (Swagger) 1.7.0** - документация API
- **JavaDoc** - документация кода

### **Тестирование:**
- **Spring Boot Test** - интеграционное тестирование
- **JUnit 5** - модульное тестирование
- **Spring Security Test** - тестирование безопасности
- **H2 Database** - in-memory БД для тестов

### **Дополнительные библиотеки:**
- **Commons IO 2.11** - работа с файлами и изображениями

##  Запуск проекта

### Требования
- Java 11
- PostgreSQL 12+
- Maven

### Настройка базы данных
```sql
CREATE DATABASE avito_db;
```
### Конфигурация
Отредактируйте application.properties:
```
spring.datasource.url=jdbc:postgresql://localhost:5432/avito_db
spring.datasource.username=ваш_логин
spring.datasource.password=ваш_пароль
```
### Сборка и запуск
```
mvn clean install
mvn spring-boot:run
```
### Доступ к приложению
- **API:** http://localhost:8080
- **Swagger UI:** http://localhost:8080/swagger-ui.html
- **H2 Console** (для тестов): http://localhost:8080/h2-console

## Функциональность
### Пользователи
- Регистрация нового пользователя
- Авторизация (Basic Auth)
- Просмотр и обновление профиля
- Смена пароля

### Объявления
- Просмотр всех объявлений (без авторизации)
- Просмотр детальной информации об объявлении
- Создание нового объявления (только USER/ADMIN)
- Редактирование/удаление своих объявлений
- Загрузка изображений для объявлений

### Комментарии
- Просмотр комментариев к объявлению
- Добавление комментария (USER/ADMIN)
- Редактирование/удаление своих комментариев

## Система прав доступа
| Роль               | Права                                                  |
| :----------------- | :----------------------------------------------------- |
| Неавторизованный   | Просмотр объявлений и комментариев                     |
| USER               | Все операции со своими объявлениями и комментариями    |
| ADMIN              | Полный доступ ко всем ресурсам                         |
